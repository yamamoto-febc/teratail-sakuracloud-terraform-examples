# クラウドでのインフラ構築を始めよう

入門編1: https://teratail.com/sakura-cloud/tutorial/2

入門編では2回に分けて以下の環境構築を行います。

- 1） 作成済みのサーバ（web-server）にNICを追加
- 2） スイッチ（192.168.0.0/24セグメント用）の作成
- 3） 2台目のサーバ（db-server）を新しく作成
- 4） マップ機能でネットワークを構成
- 5） グローバル側のサーバ（web-server）のローカル側IPアドレスを設定

入門編1では1)と2)までです。
